# sentiment-analysis-lstm
Sentiment Analysis using Bidirectional LSTM — Classifying text into Negative, Neutral, Positive. 

📍Organized dataset and preprocessed text data.

Built a Bidirectional LSTM model with embedding and dense layers.
Trained the model for 5 epochs with balanced class distribution.
Evaluated performance using accuracy, loss, classification report, and confusion matrix.
Visualized training vs validation accuracy.
Tested the model with custom sentences to check real-world predictions.
Saved the trained model and tokenizer for reuse.


Confusion Matrix: Shows balanced predictions across all three sentiment classes
Accuracy steadily improved across epochs, while validation accuracy plateaued around 85%.
Confusion matrix highlighted strong performance in Neutral and Positive classes, with slight misclassification in Negative.
Custom sentence testing showed realistic predictions with confidence scores.
